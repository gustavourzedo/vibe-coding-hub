# Cursor Setup

Cursor segue forte para:

- Figma MCP
- frontend
- design tokens
- draw.io / Mermaid / Excalidraw / PlantUML

## Restauracao

1. Copiar `mcp.template.json` para:
   - `C:\Users\gusta\.cursor\mcp.json`
2. Restaurar regras em:
   - `C:\Users\gusta\.cursor\rules\`
3. Reinstalar extensoes principais:
   - Figma
   - Frame2Code
   - Draw.io Integration
   - Mermaid Chart
   - Excalidraw
   - PlantUML
   - Tailwind CSS IntelliSense
   - Design Tokens Explorer
   - Story Explorer

Figma MCP e oficialmente bem suportado no Cursor.
